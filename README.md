# FEAST – Online Food Ordering System

## Overview

FEAST is a web-based online food ordering system developed using PHP, HTML, CSS, JavaScript, and MySQL. The application enables users to browse food items, search products, manage a shopping cart, and place orders through an intuitive interface.

## Features

- User Registration and Login
- Food Menu
- Product Search
- Shopping Cart
- Order Placement
- Responsive User Interface
- Database Connectivity using MySQL

## Technologies Used

- PHP
- HTML
- CSS
- JavaScript
- MySQL

## Project Structure

- PHP backend
- HTML pages
- CSS styling
- Product images
- MySQL database

## How to Run

1. Install XAMPP.
2. Copy the project folder to the `htdocs` directory.
3. Import the MySQL database into phpMyAdmin.
4. Start Apache and MySQL.
5. Open `http://localhost/FEAST-Online-Food-Ordering-System`.
